# 🧮 My Python Calculator

## 📖 Overview
This repository contains a simple **Python calculator** program built to perform basic arithmetic operations.

It was created as a beginner‑friendly project to practice Python fundamentals, user input, and function logic.

---

## 🎯 Purpose
This project was created to:

- Practice core Python programming concepts
- Build a console‑based calculator
- Understand functions and control flow
- Create a reusable learning project

---

## 🛠️ Features
This calculator currently supports:

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧑‍💻 Simple text‑based user interface

---

## 🧠 How It Works
The program:

1. Prompts the user to enter two numbers
2. Asks the user to choose an operation
3. Performs the calculation
4. Displays the result

It uses basic Python functions and input/output handling.

---

## 📦 Installation & Usage

### Prerequisites
- Python 3.x installed on your system

### 1. Clone the Repository
```bash
git clone https://github.com/yousri25/My-python-calculator.git
cd My-python-calculator
```

### 2. Run the Program
```bash
python main.py
```

*(Adjust filename if different)*
